# JS DOM: Il Mistero della Lampadina 💡

## 📌 Descrizione
Primo approccio alla manipolazione del DOM (Document Object Model). L'obiettivo è creare un'interfaccia utente semplice dove un'azione (il click su un bottone) scatena un cambiamento visivo immediato nella pagina.

## 🎯 Obiettivo
Implementare uno script che permetta di cambiare lo stato di un elemento immagine e il testo di un pulsante, simulando l'accensione e lo spegnimento di una lampadina.

### Focus Tecnico:
- **DOM Selection:** Utilizzo di metodi come `document.querySelector` per agganciare gli elementi HTML.
- **Event Listeners:** Utilizzo del metodo `.addEventListener('click', ...)` per intercettare l'interazione dell'utente.
- **Element Manipulation:** Modifica dinamica degli attributi di un elemento e del contenuto testuale.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** Struttura della pagina con i tag `<img>` e `<button>`.
- **CSS3:** Minima formattazione per centrare gli elementi.
- **JavaScript (ES6+):** Gestione degli eventi e modifica dinamica delle proprietà degli elementi.
